## 新建文件
```bash
$ touch filename
```

## 新建文件并且进入vim编辑
```bash
$ vi filename
```
or
```bash
$ vim filename
```

## 新建文件夹
```bash
$ madir dirname
```

#### rm -rf 目录名字
```
-r 就是向下递归，不管有多少级目录，一并删除    
-f 就是直接强行删除，不作任何提示的意思
```
