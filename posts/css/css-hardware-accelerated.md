# 使用方法
并不是所有的CSS元素属性都可以开启硬件加速，只有下面几个可以开启硬件加速： 
 
* transform  
* opacity  
* filter  

只要在元素添加这三个属性之一就可以开启。注意：CSS3的兼容性。你需要自行添加前缀。
