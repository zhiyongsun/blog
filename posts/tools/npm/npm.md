## 配置淘宝源
淘宝源
```
//配置淘宝源
npm config set registry https://registry.npm.taobao.org
//验证源
    npm config get registry
```

## update package.json
```bash
npm install -g npm-check-updates
```

检查package.json中dependencies的最新版本：
```bash
ncu
```
更新dependencies到新版本：
```bash
ncu -u
```
